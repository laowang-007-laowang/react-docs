React 环境搭建

系统要求

必需软件

  • Node.js: 14.0.0 或更高版本（推荐 LTS 版本）
  • npm: 6.0.0 或更高版本（或 yarn、pnpm）
  • 现代浏览器: Chrome 60+, Firefox 60+, Safari 12+, Edge 79+

推荐工具

  • 代码编辑器: VS Code、WebStorm、Sublime Text
  • 版本控制: Git
  • 浏览器扩展: React Developer Tools

安装 Node.js

Windows

# 方法1: 从官网下载
# 访问 https://nodejs.org/ 下载LTS版本

# 方法2: 使用包管理器
# 使用 Chocolatey
choco install nodejs

# 使用 Scoop
scoop install nodejs

macOS

# 方法1: 从官网下载
# 访问 https://nodejs.org/ 下载LTS版本

# 方法2: 使用 Homebrew
brew install node

# 方法3: 使用 MacPorts
sudo port install nodejs18

Linux (Ubuntu/Debian)

# 方法1: 使用NodeSource仓库
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 方法2: 使用包管理器
sudo apt update
sudo apt install nodejs npm

# 验证安装
node --version
npm --version

使用 Create React App 快速开始

创建新项目

# 使用 npx (推荐)
npx create-react-app my-app

# 使用 npm
npm init react-app my-app

# 使用 yarn
yarn create react-app my-app

# 进入项目目录
cd my-app

项目结构

my-app/
├── public/
│   ├── index.html
│   ├── favicon.ico
│   └── manifest.json
├── src/
│   ├── index.js          # 应用入口
│   ├── App.js            # 主组件
│   ├── App.css           # 样式文件
│   ├── App.test.js       # 测试文件
│   └── reportWebVitals.js
├── package.json          # 依赖配置
└── README.md

启动开发服务器

# 启动开发服务器
npm start
# 或
yarn start

# 浏览器会自动打开 http://localhost:3000

其他命令

# 构建生产版本
npm run build

# 运行测试
npm test

# 弹出配置（不可逆操作）
npm run eject

使用 Vite 创建项目

创建新项目

# 使用 npm
npm create vite@latest my-react-app -- --template react

配置 ESLint

安装 ESLint

# 安装 ESLint
npm install --save-dev eslint

# 初始化 ESLint 配置
npx eslint --init

ESLint 配置示例

创建 .eslintrc.js 文件：

module.exports = {
  env: {
    browser: true,
    es2021: true,
    node: true,
  },
  extends: [
    'eslint:recommended',
    '@typescript-eslint/recommended',
    'plugin:react/recommended',
    'plugin:react-hooks/recommended',
  ],
  parser: '@typescript-eslint/parser',
  parserOptions: {
    ecmaFeatures: {
      jsx: true,
    },
    ecmaVersion: 12,
    sourceType: 'module',
  },
  plugins: ['react', '@typescript-eslint'],
  rules: {
    'react/react-in-jsx-scope': 'off',
    'react/prop-types': 'off',
    '@typescript-eslint/explicit-module-boundary-types': 'off',
  },
  settings: {
    react: {
      version: 'detect',
    },
  },
};

使用 React DevTools

function MyComponent() {
  // 组件会在DevTools中显示这个名字
  MyComponent.displayName = 'MyCustomComponent';
  
  return <div>Hello World</div>;
}

// 使用 Profiler 分析性能
import { Profiler } from 'react';

function onRenderCallback(id, phase, actualDuration) {
  console.log('Component rendered:', { id, phase, actualDuration });
}

function App() {
  return (
    <Profiler id="App" onRender={onRenderCallback}>
      <MyComponent />
    </Profiler>
  );
}

项目初始化最佳实践

1. 完整的项目设置脚本

#!/bin/bash
# setup-react-project.sh

echo "🚀 创建 React 项目..."

# 创建项目
npx create-react-app $1 --template typescript
cd $1

# 安装额外依赖
echo "📦 安装额外依赖..."
npm install --save-dev prettier eslint-config-prettier eslint-plugin-prettier

# 创建配置文件
echo "⚙️  创建配置文件..."

# .prettierrc
cat > .prettierrc << EOL
{
  "semi": true,
  "trailingComma": "es5",
  "singleQuote": true,
  "printWidth": 80,
  "tabWidth": 2
}
EOL

# .env
cat > .env << EOL
GENERATE_SOURCEMAP=false
REACT_APP_VERSION=$npm_package_version
EOL

echo "✅ 项目设置完成！"
echo "运行 'npm start' 启动开发服务器"

常见问题解决

1. 端口被占用

# 查看端口占用
lsof -i :3000

# 杀死进程
kill -9 <PID>

# 使用不同端口启动
PORT=3001 npm start

2. 权限问题 (macOS/Linux)

# 修改 npm 全局目录权限
sudo chown -R $(whoami) ~/.npm

# 或使用 nvm 管理 Node.js 版本
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install node
nvm use node

3. 清理缓存

# 清理 npm 缓存
npm cache clean --force

# 删除 node_modules 重新安装
rm -rf node_modules package-lock.json
npm install
