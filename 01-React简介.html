React 简介

什么是 React？

React 是由 Facebook（现Meta）开发的一个用于构建用户界面的 JavaScript 库。它主要用于创建可重用的UI组件和管理应用程序的状态。

核心特性

1. 组件化架构

// 函数组件
function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}

// 类组件
class Welcome extends React.Component {
  render() {
    return <h1>Hello, {this.props.name}!</h1>;
  }
}

2. Virtual DOM

React使用虚拟DOM来提高性能，通过diff算法最小化实际DOM操作。

3. 声明式编程

// 声明式：描述UI应该是什么样子
function TodoList({ todos }) {
  return (
    <ul>
      {todos.map(todo => (
        <li key={todo.id} className={todo.completed ? 'completed' : ''}>
          {todo.text}
        </li>
      ))}
    </ul>
  );
}

4. 单向数据流

数据从父组件流向子组件，通过props传递。

React 生态系统

核心库

  • react: 核心库
  • react-dom: DOM渲染器
  • react-router: 路由管理
  • react-query/tanstack-query: 数据获取
  • react-hook-form: 表单处理

状态管理

  • Redux: 可预测的状态容器
  • MobX: 响应式状态管理
  • Zustand: 轻量级状态管理
  • Recoil: Facebook实验性状态管理

UI组件库

  • Ant Design: 企业级UI设计语言
  • Material-UI: Google Material Design
  • Chakra UI: 简洁的组件库
  • React Bootstrap: Bootstrap React组件

React 版本历史

版本 发布时间 主要特性
16.8 2019.02 Hooks引入
17.0 2020.10 无新特性，专注升级
18.0 2022.03 并发特性、自动批处理

适用场景

✅ 适合使用React的场景

  • 复杂的单页应用(SPA)
  • 需要频繁状态更新的应用
  • 团队协作的大型项目
  • 需要服务端渲染的应用
  • 移动端应用(React Native)

❌ 不适合使用React的场景

  • 简单的静态网站
  • 学习成本敏感的小项目
  • 需要快速原型的场景
  • SEO要求极高的营销页面

  • VS Code: 代码编辑器
